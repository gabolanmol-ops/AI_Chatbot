AI Chatbot
A simple rule-based chatbot built in Python that answers common questions about programming languages, frameworks, and AI/ML concepts.

Features
Responds to greetings and general conversation
Answers FAQs about Python, Java, C++, and AI
Explains web development frameworks (Flask, Django)
Covers basic AI/ML terms (machine learning, deep learning, automation, chatbots)
Uses a dictionary-based response system for quick lookups
